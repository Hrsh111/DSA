## üìö Pattern Quick Reference

| Pattern | When to Use | Example Problems |
|---------|-------------|------------------|
| **Two Pointers** | Sorted array, find pairs | Two Sum II, 3Sum, Container |
| **Sliding Window** | Contiguous subarray/substring | Max Subarray, Longest Substring |
| **Binary Search** | Sorted array, search space | Rotated Array, Find Minimum |
| **BFS** | Shortest path, level-order | Level Order, Number of Islands |
| **DFS** | Explore all paths, backtrack | Word Search, Clone Graph |
| **Dynamic Programming** | Optimal substructure, overlapping | Coin Change, LCS, House Robber |
| **Heap** | Top K, streaming median | Top K Frequent, Median Finder |
| **Union Find** | Connected components | Graph Valid Tree, Components |
